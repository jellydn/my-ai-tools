# Glossary

Terms and concepts used in the my-ai-tools repository.

## Terms

### AI Tools

| Term | Description |
|------|-------------|
| Claude Code | Anthropic's AI coding assistant with CLI |
| OpenCode | OpenAI-powered AI coding assistant |
| Amp | Modular's AI coding assistant |
| CCS | Claude Code Switch - profile manager for multiple providers |
| Gemini CLI | Google's AI agent for terminal |
| Codex CLI | OpenAI's command-line coding assistant |
| Kilo CLI | OpenCode-based CLI with 300+ models |
| Pi | Agentic coding workflow tool |
| GitHub Copilot CLI | Copilot in the terminal |
| Cursor Agent CLI | Cursor's background agent CLI |
| Factory Droid | Factory's AI coding agent |

### MCP (Model Context Protocol)

| Term | Description |
|------|-------------|
| MCP Server | External service that provides tools to AI assistants |
| context7 | MCP server for documentation lookup |
| sequential-thinking | MCP server for multi-step reasoning |
| qmd | MCP server for knowledge management with semantic search |
| fff | Fast file search MCP server |
| mempalace | AI memory system with palace structure |

### Hooks

| Term | Description |
|------|-------------|
| PostToolUse | Hook that runs after a tool is used (e.g., auto-format) |
| PreToolUse | Hook that runs before a tool is used (e.g., git guard) |
| Git Guard | Hook that blocks dangerous git commands |
| Auto-save hook | Hook that saves session context to memory |

### Scripts

| Term | Description |
|------|-------------|
| cli.sh | Main installation script |
| generate.sh | Export script for reverse sync |
| install.sh | Direct installer for curl-based installation |
| install.ps1 | PowerShell installer for Windows |

### Other

| Term | Description |
|------|-------------|
| Dry-run | Preview mode that shows changes without executing |
| Bidirectional sync | Ability to install configs and export back |
| Skill | Pre-configured prompt for specific tasks |
| Agent | Custom AI agent definition with instructions |
| Command | Custom slash command for CLI tools |

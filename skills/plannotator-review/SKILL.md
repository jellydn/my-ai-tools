# Plannotator Code Review

Open an interactive code review for current changes using the Plannotator UI.

## Usage

`/plannotator-review`

## Process

Run the Plannotator code review to open the interactive review UI:

```bash
plannotator review
```

Address the code review feedback. The user has reviewed changes in the Plannotator UI and provided specific annotations and comments.

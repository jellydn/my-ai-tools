# Remove AI Slop

Clean up AI-generated boilerplate, redundant comments, and unnecessary code patterns.

## What to Remove

- "This function/class/components does X" comments on obvious code
- Redundant type annotations TypeScript can infer
- Empty or unnecessary error boundaries
- Overly verbose logging statements
- Unnecessary guard clauses for impossible states
- TODO/FIXME comments without context or tracking
- Dead code paths and unused variables
